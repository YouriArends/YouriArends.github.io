Invoer Verwerking Uitvoer

Invoer

Input betekent alle gegevens die een computer of ander apparaat ontvangt van de gebruiker. Dit kan onder andere via toetsenborden, muizen, camera's, microfoons of sensoren. De invoer kan tekst, geluid, beelden of signalen zijn. Zonder input kan de computer geen nuttige informatie verwerken of taken uitvoeren.

Verwerking

Processing is de stap waarin de computer de ingevoerde gegevens bewerkt. De processor (CPU) rekent, vergelijkt en volgt de instructies van de input. Zo verandert de input in informatie die bruikbaar is. Zonder verwerking kan de computer geen zinvolle resultaten geven.

Uitvoer

Output is het resultaat dat de computer teruggeeft na de verwerking. Dit kan via tekst op een scherm, geluid via speakers of een afdruk via een printer bijvoorbeeld.
